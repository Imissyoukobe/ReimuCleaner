ReimuCleaner 博丽灵梦掉落物清理插件

一个东方Project风格的Minecraft服务器掉落物清理插件，由博丽神社的巫女博丽灵梦负责保持服务器的整洁！

功能:自动定期清理服务器内的所有掉落物/清理前后发送博丽灵梦风格的提示消息/支持自定义清理间隔、消息内容等/充满东方Project风格的文案和设计

适配版本
Minecraft Java Edition 1.21.8
Leaves服务器（兼容Paper/Spigot）

安装方法

1. 下载 `ReimuCleaner-1.0.0.jar` 文件
2. 将jar文件放入服务器的 `plugins` 文件夹中
3. 重启服务器
4. 插件将自动生成配置文件

配置说明

插件启动后会在 `plugins/ReimuCleaner/` 目录下生成 `config.yml` 配置文件，您可以根据需要修改以下配置项：

yaml文件示例：
```
清理间隔时间（秒）
clean-interval: 300

清理前预警时间（秒）
warning-time: 10

是否启用消息提示
enable-messages: true

消息前缀
prefix: "§c[博丽灵梦] §f"

博丽灵梦风格的消息列表
messages:
  warning:
    "哼，地上的垃圾越来越多了，再等一下就全部清理掉哦？"
    # 更多预警消息...
  completion:
    "呼，终于清理干净了。真是的，下次要自己收拾好啊。"
    # 更多完成消息...
```

占位符说明
`{time}`：在预警消息中显示剩余时间（秒）
`{count}`：在完成消息中显示清理的物品数量

命令说明
 `/reimucleaner reload`  重新加载插件配置  reimucleaner.admin 
 `/reimucleaner clean`  立即执行一次清理  reimucleaner.admin 
 `/reimucleaner help`  显示帮助信息  reimucleaner.use 

权限说明
`reimucleaner.use`：允许使用基本命令（默认OP）
`reimucleaner.admin`：允许使用所有命令（默认OP）

性能建议
对于小型服务器，建议将清理间隔设置为3-5分钟
对于大型服务器或高频掉落物场景，建议将清理间隔设置为1-3分钟
如果服务器性能较差，可以考虑禁用消息提示以减轻负担
